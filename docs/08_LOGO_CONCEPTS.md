# Logo Concepts — F-Hole / Frequency

## 1. Goal

`SYSTEM / SOUND` のロゴは、コントラバス奏者とAIソリューションエンジニアという二面性を、説明しすぎずに一つの形へまとめる。

中心モチーフはコントラバスのF字孔とする。

Fには次の意味を重ねる。

- **F-hole**：楽器の響きを外へ伝える開口部
- **Frequency**：音、電気、通信、ITに共通する周波数
- **Flow**：業務の流れ
- **Function**：仕組みの機能

ただし、意味をすべて図形へ詰め込まない。第一印象では静かで美しいF字孔に見え、背景を知ると技術との接続が分かる程度に留める。

## 2. Design Requirements

- 名刺上の幅12〜18mmでも認識できる
- faviconやSNSアイコンの正方形でも成立する
- 単色印刷でも形が失われない
- 細線だけに依存しない
- 音符、脳、ロボット、回路基板を直接描かない
- ネオン、グロー、サイバーパンク表現を使わない
- 木、舞台、信号を感じさせるが、楽器店やIT企業の既視感へ寄せすぎない

## 3. Concept A — Minimal Resonance

### Idea

F字孔の曲線を一筆の記号へ単純化する。

### Meaning

- 最も音楽的
- 静かで大人っぽい
- 楽器の木肌や余白と相性がよい

### Strength

- 小サイズに強い
- 単色で成立しやすい
- 名刺、焼印、箔押しへ展開しやすい

### Weakness

- AI・ITとの接続は説明がないと伝わりにくい
- 一般的なF字孔シンボルとの差別化が必要

### Best use

音楽面、名刺裏面、プロフィール写真の透かし。

---

## 4. Concept B — Signal F

### Idea

F字孔の下端を細い信号線へ接続し、小さな赤い点を一つだけ置く。

赤い点は、次のいずれにも見える曖昧な記号とする。

- 信号の現在位置
- 録音中のランプ
- 弦の振動の起点
- 課題を見つけたポイント
- 小規模な舞台の一点照明

### Meaning

音の構造とデジタル信号を、最小限の要素で接続する。

### Strength

- `SYSTEM / SOUND` の両側面が最も伝わりやすい
- 赤の差し色をブランド上の意味ある要素にできる
- Web上で、線がゆっくり描かれ赤い点が灯るアニメーションへ展開できる

### Weakness

- 赤い点が大きいと安価なテックロゴに見える
- 信号線を細くしすぎると印刷で消える

### Best use

**第一推奨案。**

サイトのメインロゴ、名刺表面、ローディング、セクション切り替えに使用する。

---

## 5. Concept C — Dual Resonance

### Idea

左右一対のF字孔を、`SYSTEM` と `SOUND`、入力と出力、相談と解決として扱う。

中央の余白をスラッシュ `/` または一本の弦として見せる。

### Meaning

二つの異なる活動を、対立ではなく響き合う関係として表す。

### Strength

- ブランド名の二項構造と合う
- シンボル単体で存在感がある
- サイトの大型ビジュアルや名刺裏面に映える

### Weakness

- 小さくすると複雑に見える
- 楽器そのものの意匠が強くなり、業務改善の印象が弱くなる可能性がある

### Best use

ファーストビューの背景、音楽セクション、大型ポスターやOGP。

---

## 6. Concept D — F / Slash Monogram

### Idea

アルファベットの `F`、F字孔、ブランド名の `/` を一つのモノグラムへまとめる。

### Meaning

- F-hole
- Frequency
- SYSTEM / SOUNDを接続するスラッシュ

### Strength

- エンジニア名刺として見たときに文字として認識しやすい
- SNSアイコン、favicon、リポジトリのアバターに強い
- ロゴタイプと組み合わせやすい

### Weakness

- 形を作り込みすぎると企業ロゴ的で硬くなる
- F字孔の自然な曲線が失われやすい

### Best use

favicon、SNSアイコン、GitHubプロフィール画像、名刺の小さなワンポイント。

---

## 7. Recommended Direction

初期案は **Concept B — Signal F** を主軸に進める。

理由：

1. F字孔の美しさを残せる
2. FrequencyとIT信号の意味を自然に追加できる
3. 赤を装飾ではなく機能的な一点として使える
4. サイト上の動きへ展開しやすい
5. 名刺表面と裏面の両方へ使用できる

Concept Aは、Signal Fから信号線と赤点を外した単色版として併用する。

Concept Cはメインロゴではなく、大型背景グラフィックとして活用する。

Concept Dは、小サイズ専用マークとしてSignal F確定後に再設計する。

## 8. Color Use

### Standard

- Background：Stage Black `#0B0B0C`
- Symbol：Varnished Maple `#C87532`
- Text：Warm Ivory `#F2EDE3`
- Point：Signal Red `#E33A2F`

### Monochrome

- Black on white
- White on black
- Single-color foil or letterpress

赤い点がない状態でもロゴとして成立することを必須とする。

## 9. Motion Direction

Web上では、次の短い動きを候補とする。

1. 暗い画面にスポットライトがゆっくり入る
2. F字孔の曲線が上から下へ描かれる
3. 下端から短い信号線が伸びる
4. 赤い点が一度だけ静かに灯る
5. `SYSTEM / SOUND` の文字が現れる

動きは0.8〜1.5秒程度。常時点滅、派手なグロー、速い波形アニメーションは禁止する。

## 10. Review Checklist

- [ ] 16px相当でも形が分かる
- [ ] 名刺上で細線が消えない
- [ ] 単色でも成立する
- [ ] 音楽教室のロゴに見えない
- [ ] AIスタートアップの既視感に寄りすぎない
- [ ] 赤の面積が全体の3％以下
- [ ] コントラバスを知らない人にも不自然な形に見えない
- [ ] 背景なし、黒背景、木目背景のすべてで確認する
